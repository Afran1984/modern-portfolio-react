# Modern React  + Tailwind CSS & ui.shadcn Component

A sleek, responsive, and modern Navigation Bar and UI template built using **React.js** and **Tailwind CSS**. This project showcases smooth interactive animations, custom Google Fonts integration, and clean component architecture.

## ✨ Features
* 🎨 **Dynamic Text & Border Animation:** Smooth Y-axis translation on buttons (hover effect) without disturbing the layout flow.
* 🔤 **Google Fonts Integration:** Global implementation of custom typography via `@import` and React Router compliance.
* 📱 **Fully Responsive:** Styled using Tailwind CSS flexbox layouts for seamless cross-device compatibility.
* 🧼 **Clean Code Structure:** Easily reusable components for landing pages or portfolio websites.

## 🛠️ Tech Stack
* **Frontend:** React.js (Vite)
* **Styling:** Tailwind CSS & ui.shadcn
* **Icons & Fonts:** Google Fonts (Montenegrin Gothic One)
